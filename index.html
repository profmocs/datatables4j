Overview

DataTables4j is an open-source taglib that allows you to quickly create HTML table in your Java/JEE based web application without even writing any HTML or Javascript code !

DataTables4j will handle all the HTML and Javascript code generation needed by the amazing DataTables jQuery plugin, depending on how you configured the JSP tags.


For what use ?
